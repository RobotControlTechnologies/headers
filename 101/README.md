# Заголовочные файлы для модулей RCML
---

При создании модуля, в cpp файл необходимо включить один из следующих заголовочных файлов:
- **_robot_module.h_** - для модулей роботов
- **_function_module.h_** - для модулей функций
- **_control_module.h_** - для модулей управления

Файл **_module.h_** должен включаться в модуль независимо от типа модуля непосредственно перед включением соответсвующего заголовочного файла.
